## Стиль дизайна

### Настроение
Дорого. Богато. Современно. Минимализм. Как сайт Apple.

### Цвета
| Элемент | Цвет |
|---------|------|
| Фон основной | Белый #FFFFFF |
| Фон секций (чередование) | Светло-серый #F5F5F7 |
| Текст основной | Тёмно-серый #1D1D1F |
| Текст второстепенный | Серый #86868B |
| Акцент (кнопки, ссылки) | Синий #0071E3 |
| Кнопка при наведении | Тёмно-синий #0077ED |

### Шрифт
- **Основной:** Inter (Google Fonts)
- **Заголовки:** Inter, жирный (700-800)
- **Текст:** Inter, обычный (400)

### Отступы и размеры
- Огромные отступы между секциями (100-120px)
- Много воздуха внутри секций
- Максимальная ширина контента: 1200px
- Скругления карточек: 16px
- Тени карточек: мягкие, едва заметные (0 4px 20px rgba(0,0,0,0.06))

### Типографика
- Заголовок Hero: 56px (desktop), 36px (mobile)
- Заголовки секций: 40px (desktop), 28px (mobile)
- Основной текст: 18px (desktop), 16px (mobile)
- Подписи: 14px

## Обязательные элементы

### Кнопки
- Скругления: 12px
- Отступы: 16px 32px
- Основная кнопка: синий фон (#0071E3), белый текст
- При наведении: плавная смена цвета (0.3s)

### Анимации
- Fade-in снизу вверх при скролле (каждая секция)
- Hover-эффекты на карточках (приподнимаются, тень усиливается)
- Плавная прокрутка к секциям
- Кнопка CTA — лёгкая пульсация тени

### Адаптивность
- Mobile-first подход
- Брейкпоинты: 768px (планшет), 1024px (десктоп)
- На мобильном: карточки в одну колонку, меню-гамбургер
- На десктопе: карточки в сетке 2x2 или 4 в ряд

### Интерактивные элементы
- Кнопка «Наверх» — появляется при скролле вниз
- Мобильное меню — гамбургер-иконка
- Плавная навигация по секциям
- Форма обратной связи (имя, email, сообщение) — визуальная, без реальной отправки

## Технические требования

- Всё в одном HTML-файле (стили и скрипты внутри)
- Чистый HTML + CSS + JavaScript — без внешних библиотек
- Шрифт Inter подключить через Google Fonts (единственное внешнее подключение)
- Адаптивная вёрстка
- Быстрая загрузка (без тяжёлых изображений)
- Семантическая разметка (header, main, section, footer)
