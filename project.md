# my_lending — Персональный лендинг Виктора Короткова

## Описание проекта

Персональный лендинг-портфолио **Виктора Короткова** — предпринимателя и вайбкодера. Сайт рассказывает о Викторе, его навыках и услугах по созданию цифровых продуктов с помощью ИИ.

## Цель сайта

- Представить Виктора как специалиста по вайбкодингу
- Показать что он умеет делать (сайты, лендинги, веб-приложения, Telegram-боты)
- Дать возможность связаться (Telegram: @VictorKorotkov)
- В будущем — портфолио с примерами работ

## Целевая аудитория

- Предприниматели, которым нужен сайт или приложение
- Люди, которые слышали про вайбкодинг и хотят заказать продукт
- Знакомые и контакты из соцсетей

## Что должно быть на сайте

1. **Первый экран** — имя, специализация, краткое описание, фото
2. **Что я делаю** — услуги (сайты, лендинги, веб-приложения, боты)
3. **Обо мне** — кто такой Виктор, опыт, подход к работе
4. **Портфолио** — примеры работ (добавится позже)
5. **Контакты** — Telegram, способ связи

## Технологии

- HTML + CSS + JavaScript (один файл или минимум файлов)
- Адаптивный дизайн (мобильные + десктоп)
- Хостинг на GitHub Pages

## Автор

**Виктор Коротков** — предприниматель с 15-летним стажем, осваивает вайбкодинг и создаёт цифровые продукты с помощью ИИ.
